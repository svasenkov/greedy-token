# Claude Desktop starter kit

Merge the MCP fragment into Claude Desktop config and paste instructions into a **Project** (recommended) or profile custom instructions.

| File | Use |
|------|-----|
| `claude_desktop_config.fragment.json` | Merge into `claude_desktop_config.json` under `mcpServers` |
| `claude_desktop_config.zero-design-system.json` | Ready-made config for `zero-design-system` monorepo (local dev venv) |
| `instructions.md` | Claude **Project → Custom instructions** (or global custom instructions) |
